# 提醒机制测试手册

## 前置
- 启动后端：
  ```bash
  ~/dev/apache-maven-3.9.12/bin/mvn -f backend/pom.xml spring-boot:run
  ```
- 启动前端：
  ```bash
  cd web
  npm run dev -- --host
  ```
- 确认根目录 `.env` 包含：
  - `VITE_API_BASE_URL=http://localhost:8080`

## 用例 A：销售端提醒弹窗
1. 左侧角色切换选择“销售”。
2. 进入“工作台”。

预期结果：
- 弹出“待录入合同提醒”弹窗。
- 文案中显示 MG 账号与金额（来自后端 `/api/reminders/unmatched`）。

## 常见问题排查
- 无弹窗：确认后端 `/api/reminders/unmatched` 有返回记录。
- 文案显示“未知”：检查后端 mock 数据是否为空。
